Etiquetas básicas de HTML

¿Qué es HTML?

Es un leguaje cifrado utilizado para el desarrollo de páginas web su siglas en ingles significan Hyper
Text Markup Language y en español Lenguaje de marcas de hipertexto.

¿Qué es una etiqueta HTML?

Son fragmentos de texto entre los sigos mayor que (<) y menor que (>) con funciones y usos específicos; Se utilizan para escribir código HTML.

Etiquetas HTML básicas

Existe una serie de etiquetas que permiten la programación en el lenguaje HTML, estas son las más usadas:

ETIQUETAS HTML
Etiqueta Utilidad o Descripción
head Es la parte superior del documento HTML donde podremos indicar el título del documento.
body Indica la parte del cuerpo del contenido de un documento HTML.
div Elemento que es usado para agrupar otros elementos y actuar como plantilla de otros
controles, nos ayuda a estructurar el documento en secciones.
a Esta etiqueta ayuda a crear un enlace a una página web. El atributo href, permite que se inserte
el enlace a conectar. E atributo target, sirve para indicar si el enlace se abrirá en una nueva o en la
misma ventana.
strong Sirve para dar énfasis a una parte en concreto del texto.
br Indica al navegador que viene un salto de línea para hacer el texto más leíble.
h1 …. h6 Sirven para insertar títulos dentro del contenido. Hay diferentes niveles de títulos, del 1
al 6. Las etiquetas h + número indican la importancia del título y del contenido.
img permite insertar y mostrar imágenes dentro del contenido.
ol – li – ul nombre.
id Las etiquetas OL y LI permiten crear listas, OL para listas ordenadas y UL para listas sin
orden, dentro de las listas, los elementos se identifican con la etiqueta LI.
p Permite agrupar texto dentro de un parágrafo. El propósito es hacer el contenido más fácil de
leer y mostrarlo de organizada.
span Permite personalizar el estilo de una palabra o texto.

Gauchat, J. D. (2012). El gran libro de HTML5, CSS3 y Javascript. Marcombo.

Atributos

¿Que es un atributo?

Es un valor adicional que configura los elementos que ajusta su comportamiento de diversas formas para cumplir los criterios de los usuarios.

Atributos HTML básicos

Eston son algunos de los atributos mas usasos en la progrmacion de paginas web.

ATRIBUTOS HTML
Atributo Valor Descripcion
id nombre Establece un identificador a la etiqueta HTML.
class nombre Establece una clase (género) a una etiqueta HTML.
lang idioma Indica el idioma del contenido de la etiqueta HTML.
traslate yes / no Indica si el contenido de la etiqueta es traducible o no.
title título Mensaje mostrado en un tooltip (aviso amarillo emergente) al mover el ratón encima.
dta-* nombre Metadatos en la propia etiqueta.
accesskey atajo Combinación de teclas que puede pulsar el usuario para activar el elemento.
dir ltr / rtl Establece la direccionalidad del texto (left to right o right to left).
style estilos Aplica propiedades CSS directamente al elemento en cuestión.

Gauchat, J. D. (2012). El gran libro de HTML5, CSS3 y Javascript. Marcombo.

Mans (2021). Lenguaje HTML, https://lenguajehtml.com/html/introduccion/atributos-comunes-html/

Valores

Son las propiedades que se le pueden asignar a los elementos como nombre, idioma, estilo entre otros.
